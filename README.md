# xunity-dev-game

Командный экзаменационный проект, представляющий собой двухмерную RPG-игру для платформы Android, разработанную с использованием **Unity 6 (6000.5.1f1)**.

Игрок исследует игровой мир, взаимодействует с NPC, сражается с противниками, получает опыт, улучшает персонажа, собирает предметы и проходит игровые уровни, завершающиеся битвой с финальным боссом.

---

# Цель проекта

Целью проекта является разработка мобильной 2D RPG-игры с использованием игрового движка Unity, реализующей основные механики жанра:

* исследование игрового мира;
* взаимодействие с NPC;
* развитие персонажа;
* автоматическую боевую систему;
* систему предметов и экипировки;
* сохранение игрового прогресса;
* пользовательский интерфейс для мобильных устройств.

# Установка и запуск проекта

## Требования

Для запуска проекта необходимо установить:

1. **Unity Hub**

2. **Unity 6 (6000.5.1f1)**
   При установке необходимо добавить модуль:

* Android Build Support

3. **Git** (если требуется клонирование репозитория)

---

## Клонирование проекта

Скачать проект можно командой:

```bash
git clone https://github.com/xomstor/xunity-dev-game.git
```

После загрузки проекта необходимо открыть его через:

```
Unity Hub → Projects → Add → Add project from disk
```

Выбрать папку:

```
xunity-dev-game
```

Не следует создавать новый Unity-проект, так как все настройки и зависимости уже находятся в репозитории.

---

# Настройка Unity

После первого открытия проекта необходимо проверить настройки:

## Project Settings → Editor

```
Version Control Mode:
Visible Meta Files

Asset Serialization Mode:
Force Text
```

## Project Settings → Player (Android)

```
Scripting Backend:
IL2CPP

Target Architectures:
ARM64
```
