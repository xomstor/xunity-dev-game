# Настройка иконок и описаний врагов в бестиарии

## Структура системы

Система состоит из двух частей:
1. **BestiaryData** — ScriptableObject, где хранятся данные врага (иконка, описание)
2. **Бестиарий в меню паузы** — отображает иконки и кнопку "Описание"

## Как добавить врага в бестиарий

### Шаг 1: Создать папку для данных

1. В Project окне создай папку: `Assets/_Project/Resources/Bestiary/`
   - Это важно! Папка должна называться **Resources** и **Bestiary**
   - Система будет искать данные именно там

### Шаг 2: Создать BestiaryData для врага

1. Правый клик в папке `Assets/_Project/Resources/Bestiary/`
2. Create → Bestiary → Enemy Entry
3. Назови файл по имени врага (например, `SpyderPi.asset`)

### Шаг 3: Заполнить данные врага

В Inspector заполни поля:

| Поле | Описание | Пример |
|------|---------|--------|
| **Enemy Id** | Уникальный ID врага | `spyder_pi_01` |
| **Enemy Name** | Имя врага (БЕЗ цифр в скобках!) | `Spyder Pi` |
| **Description** | Описание врага (большое текстовое поле) | `Быстрый враг, атакует издалека...` |
| **Enemy Icon** | PNG иконка врага | Перетащи PNG файл |
| **Base Level** | Базовый уровень | `5` |
| **Base Health** | Базовое здоровье | `100` |
| **Base Damage** | Базовый урон | `15` |

### Шаг 4: Подготовить PNG иконку

- **Размер**: 256x256 пикселей (или 128x128)
- **Формат**: PNG с прозрачностью
- **Фон**: прозрачный (Alpha = 0)
- **Стиль**: иконка должна быть узнаваема и контрастна

## Как это работает в игре

### В бестиарии (меню паузы):

1. Открой паузу (Esc)
2. Нажми кнопку "Бестиарий"
3. Каждый враг отображается с:
   - **Иконкой** (слева, 100x100 пикселей)
   - **Именем и счётчиком** (в центре)
   - **Кнопкой "Описание"** (справа)

4. Нажми "Описание" чтобы открыть панель с:
   - Большой иконкой врага (100x100)
   - Именем врага
   - Полным описанием

## Пример заполнения

### Враг: Spyder Pi

**BestiaryData файл**: `SpyderPi.asset`

```
Enemy Id: spyder_pi
Enemy Name: Spyder Pi
Description: Быстрый паук, атакует издалека. Имеет слабую защиту, но высокую скорость. Опасен в группах.
Enemy Icon: [PNG иконка паука]
Base Level: 5
Base Health: 100
Base Damage: 15
```

## Важные моменты

### ✅ Правильно:
- Enemy Name: `Spyder Pi` (без цифр)
- Enemy Name: `Goblin Warrior` (с пробелом)
- Enemy Name: `Fire Dragon` (с пробелом)

### ❌ Неправильно:
- Enemy Name: `Spyder Pi (1)` (с цифрами в скобках)
- Enemy Name: `Spyder Pi (Clone)` (с Clone)

### Почему это важно?

Когда враг умирает в игре, его имя может быть `Spyder Pi (6)` или `Spyder Pi (Clone)`.
Система автоматически очищает имя и ищет BestiaryData с именем `Spyder Pi`.

## Если иконка не отображается

1. Проверь, что файл находится в папке `Assets/_Project/Resources/Bestiary/`
2. Проверь, что **Enemy Name** совпадает с именем врага в игре (без цифр)
3. Проверь, что PNG файл назначен в поле **Enemy Icon**
4. Убей врага в игре и посмотри в бестиарий

## Если описание не отображается

1. Проверь, что BestiaryData файл находится в папке `Resources/Bestiary/`
2. Проверь, что **Enemy Name** совпадает точно
3. Заполни поле **Description** текстом
4. Нажми "Описание" в бестиарии

## Советы

- Описание может быть длинным (до 500+ символов)
- Используй переносы строк в описании для лучшей читаемости
- Иконка должна быть квадратной (256x256 или 128x128)
- Если иконки нет, будет показан серый прямоугольник

## Автоматическое обновление

После того как ты добавишь BestiaryData:
1. Убей врага в игре
2. Открой паузу → Бестиарий
3. Иконка и описание должны появиться автоматически
